# Mercadolibre frontend test -> Middleware
Este es un middleware bastante básico para la prueba de Frontend de mercadolibre.
La mayoría de recursos y esfuerzos fueron realizados en las tareas de frontend, ya que es una prueba para el mismo.

## Tecnologías
Para el desarrollo del mismo se usó:
- Node
- Express
- Axios
- Nodemon

## Instalación y uso
Para poder inicializar el middleware, ejecutar los siguientes comandos:

Producción:

``` 
  npm install
  npm start
```

Modo desarrollador: 

``` 
  npm install
  npm run dev
```

Para su uso, esta por defecto situado en `http://localhost:4000`.
